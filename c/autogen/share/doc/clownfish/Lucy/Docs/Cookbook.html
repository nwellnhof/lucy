Apache Lucy recipes

The Cookbook provides thematic documentation covering some of Apache Lucy’s more sophisticated features. For a step-by-step introduction to Lucy, see Tutorial.

Chapters

  • FastUpdates - While index updates are fast on average, worst-case update performance may be significantly slower. To make index updates consistently quick, we must manually intervene to control the process of index segment consolidation.

  • CustomQuery - Explore Lucy’s support for custom query types by creating a “PrefixQuery” class to handle trailing wildcards.

  • CustomQueryParser - Define your own custom search query syntax using QueryParser and Parse::RecDescent.
